# 📊 Habit Tracker App

A simple and intuitive **Habit Tracking Application built using Flutter**, designed to help users build better habits, stay consistent, and track daily progress efficiently.

This app focuses on productivity, minimal UI, and smooth user experience for daily habit management.

---

## 🚀 Features

- 📝 Add, edit, and delete habits  
- 📅 Track daily habit completion  
- 📊 View progress over time  
- 🔥 Streak tracking (if implemented)  
- 💾 Local storage for saving data  
- 🎯 Minimal and clean UI  
- ⚡ Smooth and responsive performance  

---

## 🛠️ Tech Stack

- **Flutter** – Cross-platform UI framework  
- **Dart** – Programming language  
- **Provider / SetState / State Management**  
- **SQLite / Hive / Local Storage**  
- **Material Design** – UI components  

---

## 📂 Project Structure

```

lib/
┣ models/        # Data models (Habit, etc.)
┣ screens/       # App screens (Home, Add Habit, Details)
┣ widgets/       # Reusable UI components
┣ services/      # Local database / logic
┗ main.dart      # Entry point

```

## ⚙️ Getting Started

### Prerequisites

- Flutter SDK installed  
- Android Studio / VS Code  
- Emulator or Physical Device  

---

### 🔧 Installation

Clone the repository
```bash
git clone https://github.com/shubham230523/habit_tracker.git
```

Navigate into project
```bash
cd habit_tracker
```

Install dependencies
```bash
flutter pub get
```

Run the app
```bash
flutter run
```
